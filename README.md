# xy-coordinate-converter  [![NPM version][npm-image]][npm-url] [![NPM Downloads][npm-downloads-image]][npm-url]
Converts XY coordinates to Lat/Lon positions

```
# install
npm install --save xy-coordinate-converter

# build
npm run build

# test
npm test
```
